# How to update Sync Folder
1. [ ] Open Files. Click Browse on bottom right tab. On My iPhone> Obsidian > Sync. Delete the entire Sync folder
2. [ ] Open iSH. type in "git clone https://github.com/TheArsonist619/Sync"
3. [ ] Go to Files, Browse>iSH>root>Sync. Press and hold Sync folder, click "move". Go to Browse>On My iPhone>Obsidian. Place "Sync" in Obsidian. 
4. [ ] Restart Obsidian.
